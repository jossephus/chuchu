Privacy Policy

Last updated: June 5, 2026

Chuchu is a local-first Android SSH client. It is designed to connect you directly to servers that you choose. Chuchu does not operate its own backend for terminal sessions, account management, analytics, or advertising.

1. Information Chuchu Stores

To provide its core features, Chuchu stores the following information on your device:

  • Server profiles such as host, port, username, transport settings, and saved preferences.
  • SSH credentials that you choose to store in the app, including passwords, key metadata, and private keys.
  • Known host fingerprints used to warn you about SSH host key changes.
  • App settings such as theme, accessory key layout, custom terminal actions, and lock preferences.
  • Encrypted backup files only when you explicitly export them.

2. How Chuchu Uses Network Access

Chuchu uses network access only to connect to servers and services that you explicitly configure or choose to reach, such as SSH, Mosh, SFTP, and Tailscale-connected hosts. Those connections are made directly from your device to the remote host.

Chuchu does not send your server list, terminal content, credentials, or backups to a Chuchu-operated server because Chuchu does not run one for the app itself.

3. Permissions

  • Internet: Required to connect to remote hosts over SSH, Mosh, SFTP, or Tailscale.
  • Network state: Used to understand connectivity status while managing sessions.
  • Foreground service: Used to keep active terminal sessions running while the app is in the background.
  • Notifications: Used to show foreground-service session notifications on supported Android versions.

4. What Chuchu Does Not Do

  • No account creation.
  • No advertising SDKs.
  • No in-app analytics or profiling.
  • No sale of personal data.
  • No remote collection of terminal content, server credentials, or SSH keys by the Chuchu developer.

5. Third-Party Services

Chuchu can connect to third-party systems only when you direct it to do so, for example your SSH server, a Mosh server process started on that host, or a Tailscale-connected device. Those systems are controlled by you or by the operator of the remote host and are outside this privacy policy.

6. Data Retention and Control

Data stored by Chuchu remains on your device unless you export it yourself. You can delete saved hosts, keys, settings, and backup files at any time. Uninstalling the app removes app-local data according to Android's normal app data handling.

7. Security

Chuchu uses SSH-related trust data such as host fingerprints to help protect against host impersonation. However, no software can guarantee absolute security, and you remain responsible for the security of the remote systems you connect to and any backups you export.

8. Children

Chuchu is not intended for children. The app is a technical tool for remote system access and is not designed to knowingly collect personal information from children.

9. Changes to This Policy

This privacy policy may be updated if Chuchu's behavior changes, for example if a future release adds cloud sync, telemetry, or other data handling features. The current version of the policy will remain published at this URL.

10. Contact

For questions about this policy or the app, open an issue at https://github.com/jossephus/chuchu/issues.
